# 📦 Estructura de Datos en MongoDB

Aprenderemos sobre cómo MongoDB organiza y almacena la información de forma flexible y diferente a las bases de datos tradicionales.

## 🧩 1. Base de Datos (Database)
En MongoDB, una **base de datos** es el contenedor principal donde se guarda toda la información.  
Puede contener múltiples colecciones.

---

## 📁 2. Colecciones (Collections)
Las **colecciones** son similares a las tablas en bases de datos relacionales, pero sin una estructura fija.

- No requieren un esquema rígido
- Pueden contener documentos con diferentes estructuras

---

## 📄 3. Documentos (Documents)
Los **documentos** son la unidad básica de datos en MongoDB.

- Se almacenan en formato **BSON** (similar a JSON)
- Contienen pares clave-valor
- Pueden tener estructuras anidadas
